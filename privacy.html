Política de Privacidad

Última actualización: 17 de junio de 2025

1. Información que Recopilamos

Datos que NO recopilamos:

  1. Información personal identificable
  2. Mensajes privados o contenido de DMs
  3. Datos de perfil de usuario
  4. Información de ubicación
  5. Datos de pago o financieros

Datos temporales que procesamos:

  1. IDs de mensajes para determinar cuáles resaltar
  2. Conteo de reacciones en mensajes públicos
  3. IDs de canales donde opera el bot
  4. Configuraciones básicas del servidor

2. Cómo Usamos la Información

  1. Procesar y resaltar mensajes populares
  2. Mantener la funcionalidad del bot
  3. Respetar la configuración del servidor
  4. Prevenir spam y abuso

3. Almacenamiento de Datos

  1. No mantenemos bases de datos de usuarios
  2. No guardamos historial de mensajes
  3. Los datos se procesan en tiempo real
  4. No hay copias de seguridad de información personal

4. Compartición de Datos

  1. No compartimos, vendemos o transferimos datos a terceros
  2. Excepto cuando sea requerido por ley
  3. Para cumplir con los términos de servicio de Discord
  4. En caso de transferencia de propiedad del bot (con aviso)

5. Seguridad

  1. Uso exclusivo de APIs oficiales de Discord
  2. Conexiones encriptadas (HTTPS/WSS)
  3. Acceso limitado solo a canales autorizados
  4. Monitoreo regular de seguridad

6. Derechos del Usuario

  1. Solicitar información sobre qué datos procesamos
  2. Remover el bot de tu servidor en cualquier momento
  3. Contactarnos sobre preocupaciones de privacidad
  4. Reportar cualquier uso indebido

7. Menores de Edad

El bot no recopila intencionalmente información de menores de 13 años. Si detectamos tal recopilación, eliminaremos los datos inmediatamente.

8. Cambios en la Política

Te notificaremos sobre cambios significativos en esta política a través de Discord o actualizaciones en esta página.

9. Contacto

Para preguntas sobre privacidad, contacta con el administrador del bot. Responderemos dentro de 48 horas.
