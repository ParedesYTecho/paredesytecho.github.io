Política de Privacidad

SkullBoard — Bot de Discord

Última actualización: 24 de junio de 2026

Esta política explica qué datos procesa y almacena el bot de Discord SkullBoard ("el bot", "nosotros") y cómo los usamos. Al añadir el bot a un servidor de Discord aceptas esta política.

1. Qué datos almacenamos (y cuáles no)

SkullBoard sigue un principio de minimización de datos: solo guardamos identificadores numéricos de Discord y contadores necesarios para funcionar. Nunca guardamos el texto de los mensajes.

✅ NO almacenamos

  • El contenido o texto de los mensajes
  • Nombres de usuario, apodos o avatares
  • Mensajes directos (DMs) privados
  • Correos, datos de pago o ubicación
  • Cualquier dato de perfil personal

📦 SÍ almacenamos

  • ID del mensaje destacado
  • ID del autor del mensaje (numérico)
  • ID del servidor y de los canales
  • Número de reacciones 💀 recibidas
  • Configuración del servidor y preferencias

En detalle, almacenamos los siguientes registros:

DatoFinalidad

El contenido de un mensaje destacado se reenvía en el momento al canal del skullboard de tu servidor (dentro de Discord), pero no se guarda en nuestra base de datos. Se procesa de forma transitoria en memoria para construir el mensaje destacado.

2. Intents privilegiados de Discord

Para funcionar, el bot utiliza los siguientes intents de Discord:

  • Message Content Intent: para leer el contenido del mensaje que alcanza el umbral de 💀 y reconstruirlo como destacado, y para detectar los comandos de texto con prefijo. El contenido se usa de forma transitoria y no se almacena.
  • Server Members Intent: para resolver el apodo y avatar actuales de los autores destacados al mostrar rankings y estadísticas. Solo consultamos a los autores concretos de mensajes destacados; no exportamos ni recopilamos la lista de miembros.

El bot no utiliza el Presence Intent ni rastrea la actividad o el estado de los usuarios.

3. Dónde se almacenan los datos

Los datos se guardan en una base de datos alojada en Supabase (PostgreSQL) y el bot se ejecuta en un proveedor de alojamiento en la nube. Todas las conexiones están cifradas (HTTPS/WSS). El acceso a la base de datos está restringido al administrador del bot.

4. Compartición de datos

  • No vendemos, alquilamos ni compartimos datos con terceros con fines comerciales o publicitarios.
  • Los datos se procesan únicamente a través de proveedores de infraestructura (Discord y Supabase) necesarios para el funcionamiento del bot.
  • El contenido de los mensajes no se utiliza para entrenar modelos de aprendizaje automático ni de inteligencia artificial.
  • Solo divulgaríamos datos si la ley lo exigiera.

5. Conservación y eliminación

  • El registro de un mensaje destacado se elimina automáticamente cuando el mensaje original se borra en Discord.
  • Los administradores pueden borrar todos los datos del skullboard de su servidor con el comando de reset.
  • Al expulsar al bot del servidor puedes solicitar la eliminación de los datos restantes asociados a ese servidor (ver Contacto).

6. Tus derechos y opciones

  • Exclusión de canales: los administradores pueden añadir canales a la lista de ignorados para que nunca se procesen.
  • Notificaciones: cualquier usuario puede desactivar los mensajes directos del bot con el comando de notificaciones.
  • Acceso y borrado: puedes solicitar saber qué datos guardamos sobre ti o pedir su eliminación escribiéndonos.

7. Menores de edad

El bot está pensado para usarse conforme a los Términos de Servicio de Discord, que requieren una edad mínima. No recopilamos intencionadamente datos de menores de 13 años. Si detectamos tales datos, los eliminaremos.

8. Cambios en esta política

Podemos actualizar esta política. Los cambios relevantes se reflejarán en esta página con una nueva fecha de actualización.

9. Contacto

Para preguntas sobre privacidad o solicitudes de datos, escribe a ferociouspyt1@gmail.com. Responderemos en un plazo de 48 horas.
